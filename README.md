# IIC2433-Data-Mining

El Ramo IIC2433 es un ramo opcional para la licenciatura de ingeniría matemáticas en la Pontificia Universidad Católica. 

Se cursó este ramo el primer semestre del 2024 cuando me encontraba en mi cuarto semestre de licencitura.

## Proyectos
Este curso corresponde a 4 tareas individuales y un proyecto final grupal. Las tareas se centran en modelar, entender y proyectar distintos datasets usando varias técninas.

Lamentablemente, solo tengo acceso a una vista preliminar y no final del proyecto. Este tiene como objetivo usar técninas del curso para la clasificación de vinos dados sus características bioquímicas. La mayor dificultad fue el desbalance del dataset.
